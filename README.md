# BPRC-LaTeX-Examples
This repository contains the powerpoint presentation used for the Building Performance Research Center (BPRC) Seminar at Carleton University, as well as example code used for some of the illustrations shown in the presentation. All .tex files are accompanied by pdf files with the image outputs, so you can preview what the code does. 

A few notes: DemonstrationInput.tex has the same output as Demonstration.tex, and can be seen by opening Demonstration.pdf. The only difference is that DemonstrationInput.tex plots by reading from a .csv file (also in the folder as data.csv, which you will need to download into the same folder as DemonstrationInput.tex if you want it to run). Capture.png needs to be downloaded (in the same folder) as "Two Pareto - Circle.tex" for it to work, as the .tex file imports an external image.

All of these pictures have been created by me for coursework, demonstrations, and presentations. None of the above appear in published research, though some MAY appear within my PhD proposal/thesis. As such, if you're using the code as-is, kindly cite the repository.
