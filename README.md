# BPRC-LaTeX-Examples
This repository contains the powerpoint presentation used for the Building Performance Research Center (BPRC) Seminar at Carleton University, as well as example code used for some of the illustrations shown in the presentation. All .tex files are accompanied by pdf files with the image outputs, so you can preview what the code does. The commit messages give some useful info as well to help you understand what you're looking at.

A few notes: DemonstrationInput.tex has the same output as Demonstration.tex, and can be seen by opening Demonstration.pdf. The only difference is that DemonstrationInput.tex plots by reading from a .csv file (also in the folder as data.csv, which you will need to download into the same folder as DemonstrationInput.tex if you want it to run). Capture.png needs to be downloaded (in the same folder) as "Two Pareto - Circle.tex" for it to work, as the .tex file imports an external image.

All of these pictures have been created by me for coursework, demonstrations, and presentations (academic and otherwise). None of the above appear in published research, though some MAY appear within my PhD proposal/thesis. As such, if you're using the code as-is, kindly cite the repository. While LaTeX is pretty consistent across IDE's and versions, I will note that all of these examples were created on TeXstudio, Version 4.7.1.

P.S. As much as I think these examples are good, I'm 1000% sure that many/all of these .tex files could be trimmed down, simplified, and possibly written better than how I did them (especially the TikZ ones). These examples do not constitute documentation-level examples in terms of code-cleanliness. Several have useless commented-out lines and packages unnecessarily carried over from older .tex files. I did not put in any effort into cleaning these files up, but they're still quite readable.

Good documentation for pgfplots and tikz can be found in:
